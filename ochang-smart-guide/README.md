# 오창휴게소 스마트 안내 샘플

기본 CMS의 URL/WebView 메뉴에서 실행할 수 있도록 만든 정적 웹 샘플입니다.

## 포함 기능

- 본관 푸드코트 / 24시간 로봇 푸드코트 / 커피·아이스크림 구역 분류
- 메뉴별 예상 제조시간을 반영한 빠른 주문 가능 메뉴
- 라면로봇 4종 및 한식로봇 기본·지역 특화 메뉴 조회
- 구역별 예상 대기시간 및 혼잡도
- QR 모바일 주문 안내
- 주문번호별 제조·픽업 현황
- 본관·별관·특별관 시설 안내
- 큰 글씨 및 다국어 선택 UI
- 데모 데이터 새로고침

## 실행

```bash
python3 -m http.server 8765
```

브라우저에서 `http://127.0.0.1:8765`를 엽니다.

GitHub Pages에서는 저장소 루트의 `index.html`을 시작 페이지로 사용합니다.

## H-ROS 연동 지점

현재 `app.js`의 `stores`와 `orders` 배열이 데모 데이터를 제공합니다. 실제 적용 시 다음 API 응답으로 교체합니다.

- `GET /api/stores`: 코너·로봇 운영 상태, 예상 대기시간, 메뉴, 품절 여부
- `GET /api/orders/public`: 공용 화면용 주문번호, 제조 상태, 픽업 위치
- `GET /api/facilities`: 시설 위치와 운영 상태
- 모바일 주문 URL: `renderOrder()`의 QR 연결 주소

공용 안내로봇 화면에는 개인정보를 표시하지 않고 주문번호와 픽업 정보만 노출하는 것을 전제로 합니다.
